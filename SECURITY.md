# Security

Royale is an experimental prototype. It is not production software and has not undergone a security review.

If you find a vulnerability, please avoid opening a public issue with exploit details. Use GitHub private vulnerability reporting or GitHub Security Advisories if enabled for the repository.

If private reporting is not available, open a minimal public issue asking for a private contact path, without including sensitive details, proof-of-concept code, credentials, tokens, or exploit instructions.

Please include enough context to reproduce and assess the issue, such as affected commit, platform, command line, network exposure, logs with secrets removed, and expected versus observed behavior.
