Integrate OpenTelemetry into the dedicated server for configurable structured logs, metrics, and traces with OTLP export, without making observability required for normal local development.

## Implementation Notes

- Added OpenTelemetry `1.16.0` and `OpenTelemetry.Exporter.OpenTelemetryProtocol` `1.16.0` to central package management.
- Added `RoyaleTelemetry` in `Royale.Diagnostics` as the dedicated server telemetry bootstrap.
- Server telemetry defaults to service name `royale-server`, activity source `Royale.Server`, and meter `Royale.Server`.
- ZLogger console output remains enabled for all server runs.
- OTLP log, trace, and metric export are created only when `OTEL_EXPORTER_OTLP_ENDPOINT` is set to a non-empty absolute URI and `OTEL_SDK_DISABLED` is not `true`.
- `OTEL_SDK_DISABLED=true` suppresses OTLP exporter/provider setup even when an endpoint is configured.
- Standard OpenTelemetry exporter and SDK environment variables remain available; the bootstrap only gates setup and applies a 1 second local-development default for exporter/processor shutdown timeouts when timeout env vars are absent.
- `Royale.Server` now wraps startup/run/shutdown in a bounded `royale.server.run` activity with startup tags.
- Updated `architecture/observability` with the v1 enablement behavior.

- `dotnet build Royale.slnx -m:1 --no-restore`
- `dotnet test Royale.slnx -m:1 --no-restore`
- `dotnet run --project src/Royale.Server/Royale.Server.csproj -- --map graybox --run-ticks 5`
- `OTEL_EXPORTER_OTLP_ENDPOINT=http://localhost:4317 dotnet run --project src/Royale.Server/Royale.Server.csproj -- --map graybox --run-ticks 5` outside the sandbox so UDP bind is permitted; no collector was running and startup still completed.

## Non-Goals

- No gameplay metrics, per-player metrics, packet metrics, or structured gameplay events were added; `OBS-002` owns those.
- No client OpenTelemetry wiring was added.
- No Kubernetes manifests, OpenTelemetry Collector config, Prometheus, Loki, Tempo, Grafana dashboards, or Grafana actions were added.
